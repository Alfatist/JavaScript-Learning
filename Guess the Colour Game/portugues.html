COMO JOGAR

RGB é um acrônimo para "Red Green Blue" (Vermelho, verde e azul); o computador combina essas três cores para gerar todas as outras cores.
O computador usa no total 255 tonalidades delas.

Exemplo: RGB(255, 0, 0), 255 vermelho + 0 verde + 0 azul = vermelho.
RGB(255, 0, 255), 255 vermelho + 0 verde + 255 azul = roxo.
