HOW TO PLAY

RGB is an acronym for "Red Green Blue"; the computer combines these three colors to generate all the other colors.
The computer uses a total of 255 shades of these colors.

Example: RGB(255, 0, 0), 255 red + 0 green + 0 blue = red.
RGB(255, 0, 255), 255 red + 0 green + 255 blue = purple.
